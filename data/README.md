# 📈 MarketMock - Trading Simulator

MarketMock е уеб базиран симулатор за тестване на търговски стратегии (backtesting). Изграден е с **Python** и **Django**, като използва стриктни принципи на Обектно-ориентираното програмиране (OOP) и Design Patterns (Factory Pattern).

## 🚀 Функционалности

* **Източници на данни:** Поддържа четене на исторически цени от локален `CSV` файл или динамично изтегляне чрез външно API (Binance).
* **Мързеливо зареждане (Lazy Evaluation):** Данните се обработват чрез Python генератори (`yield`), което позволява симулиране на огромни масиви от данни без претоварване на паметта.
* **Търговски стратегии:** 
  * *Simple Strategy* (Купува при спад, продава при ръст)
  * *Momentum Strategy* (Следи тренда)
  * *Always Buy Strategy* (Агресивно инвестиране)
* **Финансови метрики:** Автоматично изчисляване на Възвръщаемост (ROI) и Максимален спад (Max Drawdown).
* **Уеб интерфейс:** Чист и интуитивен UI, базиран на Bootstrap 5.
